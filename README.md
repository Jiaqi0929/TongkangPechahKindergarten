# Tongkang Pechah Kindergarten Website

This project is a **Kindergarten Information Website** designed to provide parents and visitors with information about the kindergarten, including its background, activities, facilities, and contact details.

The website presents the kindergarten in a simple and user-friendly interface so visitors can easily learn about the school.

---

# Live Website

You can view the website here:

**Main Website:**
https://tongkang-pechah-kindergarten.html-5.me/index.php

---

# Features

* Homepage introducing the kindergarten
* About page with kindergarten background
* Activities page showing student activities
* Facilities page describing school facilities
* Gallery page displaying photos
* Contact page with communication information
* Responsive and simple website layout

---

## Notes

- This project is connected to a private MySQL database used by the Tongkang Pechah Kindergarten website.
- The database file is not included in this repository to protect the organization's data.
- Therefore, the project cannot be fully run locally without the database configuration.
- Users are recommended to view the live deployed website instead.
- Website content such as information, activities, and gallery data are retrieved dynamically from the database.
- The project demonstrates the integration of frontend and backend web technologies.

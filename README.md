# Tongkang Pechah Kindergarten Website

This project is a Kindergarten Information Website designed to provide parents and visitors with information about the kindergarten, including its background, activities, facilities, and contact details. The website presents the kindergarten in a simple and user-friendly interface so visitors can easily learn about the school. It demonstrates full-stack development with PHP and MySQL integration.

## 🔗 Live Demo
[Main Website](https://tongkang-pechah-kindergarten.html-5.me/index.php)

## ✨ Features
- Homepage introducing the kindergarten
- About page with kindergarten background
- Activities page showing student activities
- Facilities page describing school facilities
- Gallery page displaying photos
- Contact page with communication information
- Responsive and simple website layout
- Dynamic content retrieved from a MySQL database

## 🛠️ Technologies Used
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** PHP
- **Database:** MySQL

## 📁 Project Structure
| File / Folder | Description |
|---------------|-------------|
| index.php | Main homepage of the kindergarten website |
| about.php | Information about the kindergarten |
| activities.php | Displays kindergarten activities |
| facilities.php | Shows available facilities |
| gallery.php | Photo gallery of the kindergarten |
| contact.php | Contact information page |
| styles.css | Website styling and layout |
| scripts.js | JavaScript for interactive features |

## 💡 What I Learned
- Integrating a frontend interface with a backend database
- Building dynamic pages that pull content from MySQL
- Creating a user-friendly experience for non-technical users (parents)
- Structuring a multi-page website with consistent navigation

## 📝 Note
This project is connected to a private MySQL database used by the Tongkang Pechah Kindergarten website. The database file is not included in this repository to protect the organization's data. Therefore, the project cannot be fully run locally without the database configuration. Users are recommended to view the live deployed website instead.
